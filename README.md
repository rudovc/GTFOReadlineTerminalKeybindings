# Readline Terminal Keybindings

BepInEx mod for GTFO that adds readline keybindings to the in-game terminal.

## Bindings

| Key | Action |
|-----|--------|
| Ctrl+A | Move to start of line |
| Ctrl+E | Move to end of line |
| Ctrl+U | Clear line |
| Ctrl+W | Delete word behind cursor |
| Ctrl+K | Delete from cursor to end of line |
| Alt+F | Move forward one word |
| Alt+B | Move backward one word |

## Requirements

- [BepInEx](https://docs.bepinex.dev/) (IL2CPP, BepInEx 6)
- [MTFO](https://thunderstore.io/c/gtfo/p/dak/MTFO/)

## Install

Copy `ReadlineTerminalKeybindings.dll` into `BepInEx/plugins/`.
